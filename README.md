## O que é o DevRoast?

DevRoast é uma ferramenta de revisão de código que avalia seu código de forma... digamos, **honesta**. Você cola seu código, e em troca recebe uma avaliação sem filtros sobre a qualidade do seu código — com direito a sarcasmo e roast mode.

### Funcionalidades Principais

- **Envio de Código** - Cole seu código e receba uma análise instantânea
- **Roast Mode** - Ative o modo sarcasmo para avaliações ainda mais... condimentadas
- **Leaderboard** - Veja os piores códigos da internet ranqueados por vergonha
- **Preview de Código** - Destaque de sintaxe bonito com tema escuro

### Tech Stack

- Next.js + React
- Tailwind CSS
- Shiki para syntax highlighting
- Radix UI

## Como Executar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/devroast.git

# Instale as dependências
npm install

# Execute o projeto
npm run dev
```

Acesse http://localhost:3000
